# Java_Final_Project
Java Programming Fundamental Final Project (Hyeonji Jo 148697)

This is a turn-based business decision game.
The player manages TechCorp, starts a project, assigns employees, and works over turns until the project is finished.

Implemented features:
- Company model
- Employee inheritance hierarchy
- Developer, Tester, Manager roles
- Project model
- ProjectStatus enum
- Turn-based game loop
- Console menu
- Input validation
- Basic exception handling
